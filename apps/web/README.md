# Neural Learning Universe — Next.js Frontend

This is the dedicated Next.js (App Router, TypeScript, Tailwind, Shadcn UI, Framer Motion) premium frontend for Neural Learning Universe.

## Folder Structure

- `/app` — All pages and routing (App Router: layout, home, dashboard, chat, pdf, quiz, notes, achievements, settings, auth)
- `/components` — Reusable components (UI, layout, chat, pdf, quiz, animations)
- `/styles` — Tailwind global CSS
- `/public` — Assets (logo, future screenshots)
- `/types` — TypeScript types
- `/lib` — Helper code for API, utils, hooks

## Getting Started

First, install dependencies:
```sh
cd apps/web
pnpm install # or npm/yarn
```

Then start the dev server:
```sh
pnpm run dev # or npm run dev
yarn dev
```

## Features
- App shell
- Premium layout (glassmorphism, gradients, light/dark, animation-ready)
- Route scaffolds for all major features
- Modern UI libraries installed (Shadcn UI, Lucide, Framer Motion)
- Production-ready for full-stack integration
